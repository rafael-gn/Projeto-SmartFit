📌 Sobre o projeto

Projeto Front-End desenvolvido em React inspirado em plataformas fitness, com foco em responsividade, componentização e experiência do usuário. A aplicação utiliza conceitos modernos de desenvolvimento web para construção de interfaces dinâmicas, reutilização de componentes e adaptação para diferentes tamanhos de tela, reforçando práticas de organização e escalabilidade no Front-End.

🚀 Tecnologias utilizadas

- React.js

- JavaScript

- CSS3

- HTML5

⚙️ Funcionalidades

- Interface moderna e responsiva

- Organização de componentes reutilizáveis

- Navegação dinâmica

- Estruturação de layout responsivo

= Experiência otimizada para diferentes dispositivos
